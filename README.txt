Project: XSearch
Sub-Project: Infinity War 

The purpose of this project is to create a benchmarking suite capable of pushing search engine libraries to thier limit to find bottlenecks in eaach library. There are 2 main areas of testing for these libraries which are Indexing and the other is Search. Each main category is split into sub categories targeting specific components.
 
Indexing
	(Throughput, Size, Latency)
Search
	(Latency, Throughput).

Current Libraries include Lucene, Lucene++, and XSearch(Custom Library).
